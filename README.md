# CreditPathAI

CreditPathAI is a machine learning project developed to predict loan default risk using structured financial and demographic data. The objective is to help financial institutions assess borrower risk and make informed lending decisions.

---

##  Problem Statement

Financial institutions face significant risk when approving loans. Predicting whether a borrower will default can:

- Reduce financial losses
- Improve credit risk assessment
- Support data-driven lending decisions

This project builds a predictive model using historical loan data.

---

##  Dataset

The dataset used in this project:

- `Loan_Default.csv` – Original dataset
- `Loan_Default_Preprocessed.csv` – Cleaned and transformed dataset

The dataset contains borrower details such as:

- Demographic information
- Financial attributes
- Loan characteristics
- Default status (Target variable)
