Loan Dataset – Data Preprocessing

Overview
This module focuses on preprocessing the loan dataset to make it suitable for further analysis and model development.

Preprocessing Steps Performed
- Handling missing values
- Removing duplicate records
- Encoding categorical variables
- Feature selection
- Data cleaning and formatting

Output
The cleaned dataset is saved as:
Loan_cleaned.csv

Files Included
- preprocess_loan.py → Python script for preprocessing
- Loan_cleaned.csv → Processed dataset
- requirements.txt → Required libraries

Tools Used
- Python
- Pandas
- NumPy
